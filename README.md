# Trabajo_Practico_Integrador-Programacion-1
# Sistema de Gestión de Países

# Descripción del proyecto

Aplicación desarrollada en Python para administrar información de países mediante archivos CSV. El sistema permite agregar, actualizar, buscar, filtrar, ordenar y obtener estadísticas sobre los países registrados.

# Instrucciones de ejecución

1. Clonar o descargar el repositorio.
2. Abrir el proyecto en Visual Studio Code.
3. Verificar que exista el archivo CSV dentro de la carpeta `datos`.
4. Ejecutar python main.py
